Оформление на сайтах(Украина)

Для оформления новостей и информационных страниц на https://errors-seeds.com.ua/ желательно использовать палитру:

#65bd00
#de5d48
#ffffff

Для оформления списка сортов использовать код:

<div class="d-flex flex-row flex-wrap"> <a href="ссылка на сорт" class="blog_product"> <img src="путь к картинке"> <span>Название сорта</span> </a> </div>

Стили прописываются по принципу _название стиля_ : _значение стиля_ (Пример color: #fff)

Что бы перечислить несколько стилей, раздели их точкой с запятой (Пример color: #fff ; text-transform: uppercase;)

Заливка фона(background:)

<span style="background: #000000;"></span>

Границы(border:)

<span style="border: 2px dotted #000000">dotted (точки)</span> <span style="border: 2px solid #000000">solid (линия)</span>

Цвет текста(color:)

<span style="color: #000000"></span>

Шрифт(font-family:)

<span style="font-family:Times New Roman, Times, serif "></span> <span style="font-family:Arial Black, Arial Black, Gadget, sans-serif "></span>

Размер шрифта(font-size:)

<span style="font-size: 16px"></span> <span style="font-size: 18px"></span>

Жирность шрифта(font-weight:)

<span style="font-weight: 100">нормальный</span> <span style="font-weight: 600">жирный</span>

Выравнивание текста(text-align:)

style="text-align: center;" style="text-align: left;" style="text-align: right;" style="text-align: justify;"

Скрытие ссылок

На всех сайтах присутствует скрипт для скрытия исходящих ссылок. Для того что бы сктыть ссылку нужно добавить класс hidli, а путь записать в rel="" :

<a href="#" target="_blank" class="hidli" rel="https://errors-seeds-p.com/contact/"></a>

Исправление табов

На клонах 2.0 в разделе "о нас" и "сотрудничество", табы, в ходе их редактирования не через код, а в редакторе, часто ломаются. Для их исправления необходимо:

- Открыть код
- Прокрутить до самого начала

- И убрать класс active у всех li

<li class="nav-item active">...</li> <li class="nav-item">...</li>
